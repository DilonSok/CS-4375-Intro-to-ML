    pip install pandas
    pip install numpy
    pip install matplotlib
    pip install scikit-learn
    pip install seaborn

2. After installing the packages, download my source files to a directory of your choice

3. After installing the source code, run the file respective to your IDE

        and run the file by using the command 'python hw1.py' without the single quotes
